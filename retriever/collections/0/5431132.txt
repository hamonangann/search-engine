Tugas berikut menjelaskan beberapa skenario konfigurasi logging: 1 Hentikan pesan DEBUG dan INFO agar tidak masuk ke file log. 2 Izinkan pesan tingkat INFO dari subsistem HTTP dipublikasikan ke file log, tetapi tidak ke standar keluar. 3 Tentukan bahwa penangan menerbitkan pesan yang tingkat keparahan PERINGATAN atau lebih tinggi.
