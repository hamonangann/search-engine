Kecepatan cahaya adalah 300.000.000 meter per detik, tetapi kecepatan suara hanya sekitar 340 meter per detik. Jika Anda menonton kembang api yang berjarak sekitar satu kilometer (1000 meter), cahaya hanya membutuhkan waktu 3 sepersejuta detik untuk mencapai Anda. Suara membutuhkan waktu sekitar 3 detik. Anda dapat mengetahui berapa kilometer jauhnya kembang api meledak dengan mulai menghitung detik segera setelah Anda melihat ledakan.
