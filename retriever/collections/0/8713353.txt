Berikut adalah panduan tempat untuk menonton kembang api di atas Sungai Delaware. Untuk lebih banyak lagi, lihat panduan lengkap kami tentang hal-hal yang harus dilakukan di malam hari selama akhir pekan Festival Kapal Tinggi. Pada hari Sabtu, 27 Juni, kembang api meledak di atas Sungai Delaware. Ingat, pada tanggal 4 Juli, kembang api hanya ada di atas Parkway.
