Ekstensi File DOCX. File dengan ekstensi file DOCX dapat dibuka dengan pengolah kata Jarte gratis. Ekstensi file DOCX mewakili file dokumen yang dibuat dengan pengolah kata Microsoft Word 2007. Ekstensi file DOCX adalah format baru yang dibuat oleh Microsoft dan pertama kali digunakan di Word 2007. DOCX berbeda dari ekstensi file DOC karena DOCX adalah format file dokumen berbasis XML 1 non-proprietary.
