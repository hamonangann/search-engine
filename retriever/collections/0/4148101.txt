Ini adalah jumlah blok yang akan ditulis oleh penulis basis data di setiap batch. Standarnya adalah 512 atau db_files * db_file_simultaneous_writes / 2 hingga batas seperempat nilai db_cache_size. Parameter Paralel Tersembunyi. Parameter paralel tersembunyi yang paling penting adalah _parallelism_cost_fudge_factor. Parameter ini mengatur pemanggilan Oracle Parallel Query (OPQ) oleh pengoptimal SQL berbasis biaya saat parallel_automatic_tuning=TRUE. Dengan menyesuaikan parameter ini, ambang batas untuk memanggil kueri paralel dapat dikontrol.
