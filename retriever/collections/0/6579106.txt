1. Mengunduh dan menginstal perangkat lunak JDK. Untuk menulis dan menjalankan program Java, Anda perlu menginstal program perangkat lunak yang disebut Java SE Development Kit (atau disingkat JDK, dan SE berarti Edisi Standar). JRE (Java Runtime Environment): adalah inti dari platform Java yang memungkinkan menjalankan program Java di komputer Anda.
