Run dipanggil untuk memulai aplikasi WPF. Jika Anda mendefinisikan Aplikasi Anda menggunakan markup, atau markup dan kode di belakang, Run akan dipanggil secara implisit. Namun, jika Anda mendefinisikan Aplikasi menggunakan kode, Anda perlu memanggil Run secara eksplisit. Saat Run dipanggil, Application melampirkan instance Dispatcher baru ke utas UI. Selanjutnya, metode Run objek Dispatcher dipanggil, yang memulai pompa pesan untuk memproses pesan windows. Akhirnya, objek Dispatcher memanggil metode OnStartup objek Aplikasi untuk meningkatkan acara Startup.
